- Para instalar nuevos paquetes con (pnpm)
```
pnpm add nombre-paquete
```
- Si queremos instalar swagger
-  npm install --save @nestjs/swagger
con pnpm:
```
pnpm add @nestjs/swagger
pnpm approve-builds
```

Para configurar variables de entorno:
```
pnpm add @nestjs/config
```

- Para trabajar con Bases de datos (TypeORM)
```
pnpm add @nestjs/typeorm typeorm pg
```
- instalar dotenv para variables de entorno
```
pnpm add dotenv
```

- Instalar para validacion
```
pnpm add class-validator class-transformer
```

- Para cifrar contraseñas
```
pnpm add bcrypt
pnpm approve-builds
```

- Para JWT

```
pnpm add @nestjs/jwt
```

- Para Archivos (imagenes)

```
pnpm add multer
pnpm add -D @types/multer
```
